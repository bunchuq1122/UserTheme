# Liven up your profile with songs!

***how to add song***
1. Open Geode MOD Settings
2. Put SONG ID into **Profile Song ID** Field
3. Open your profile at least once
4. and **Done**

Share your theme with everyone! *(thx for UserDataAPI)*
